## Setup

* git clone https://github.com/deVinnnie/devlabs-2018-04-20.git
* Install Gradle (https://gradle.org/install/#manually)
* Add `export PATH=$PATH:"~/Documents/bin/gradle-4.7/bin"` to your `~/.bashrc`
* GitBash -> `cd projectdir` -> `gradle ...`
